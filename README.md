## Features

- OAuth flow for Spotify & Genius
- Load saved tracks
- Progress bar (visual)
- Lyrics with web crawling
- BPM

## Future Features

- Ability to seek track to progress
- Better lyrics searching logic
- Various user/accounts
- Better UI design
- Search for tracks
- Add / Delete tracks
- Manage playlists
- Waveform visualization
